Macro to determine S\+DL version program was compiled against. 

This macro fills in a \hyperlink{struct_s_d_l__version}{S\+D\+L\+\_\+version} structure with the version of the library you compiled against. This is determined by what header the compiler uses. Note that if you dynamically linked the library, you might have a slightly newer or older version at runtime. That version can be determined with \hyperlink{_s_d_l__version_8h_a4204ecbb600f51ee50515979e60bf8f3}{S\+D\+L\+\_\+\+Get\+Version()}, which, unlike \hyperlink{_s_d_l__version_8h_a5bced26798f24e0eac0555adfd8d9689}{S\+D\+L\+\_\+\+V\+E\+R\+S\+I\+O\+N()}, is not a macro.

Get the version of S\+DL that is linked against your program. 

If you are linking to S\+DL dynamically, then it is possible that the current version will be different than the version you compiled against.
