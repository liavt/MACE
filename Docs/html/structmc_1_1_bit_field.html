Detailed Description

template<typename T>
struct mc::BitField< T >

Similar to

std::bitset.

Holds

T

and allows for bit manipulation via methods. Can set individual bits. Additionally, it overrides almost every operator to operate on it's wrapped value, to allow for you to manually manipulate it.

To construct it, you can simply call

BitField<Byte> field = 0b0000000.

There are constructors for those who want to use them.
