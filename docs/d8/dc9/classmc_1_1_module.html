Detailed Description

Abstract class used for doing a task in MACE.

Plugged into System via addModule().

Member Function Documentation

virtual void mc::Module::destroy ( )
protectedpure virtual

Called when System#terminate is called and this Module is registered.

Use this to clear memory, close streams, or clean up anything that needs to be cleaned up.

Should only be called before the program is closed.

Implemented in mc::gfx::EntityModule, mc::gfx::GraphicsModule, mc::win::WindowModule, and mc::net::NetworkModule.

virtual std::string mc::Module::getName ( ) const
protectedpure virtual

Override this function and return a UNIQUE name for this `Module.

`

The name for your Module is used in comparison, so make sure it is as unique as possible.

It is akin to a hashcode.

Implemented in mc::gfx::EntityModule, mc::gfx::GraphicsModule, mc::win::WindowModule, and mc::net::NetworkModule.
