void mc::Preprocessor::addInclude ( Include &  incl)

Adds an Include to this preprocessor.

Any Include's in this Preprocessor will be used when #include is encountered.

Because this function takes in a reference, the Include provided must remain owned by the program when preprocess() or preprocessTokens() is called.
