void mc::System::removeModule ( Module &  m)
static

Remove a Module by reference.

It will find the Module based on their getName() function, so if there are multiple Modules with the same name, this function may produce unexpected behavior.

Parameters
mReference to Module in the buffer.
Exceptions
ObjectNotFoundInArrayif the referenced Module doesn't exist.
