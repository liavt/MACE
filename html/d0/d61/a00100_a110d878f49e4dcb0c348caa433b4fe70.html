Index mc::System::addModule ( Module &  m)

Register a Module.

Registered Modules will have update(), init(), and destroy() are respectivaly called when System.update(), System.init(), and System.destroy() are called.

Parameters
mReference to a Module. If the Module referenced leaves scope or gets deallocated, a read access violation will occur.
Returns
Location of the Module in the buffer. This index can be used in various other methods.
