Detailed Description

Special object that is used for asynchronous queries of information from the GPU.

Even though it extends Object it does not implement the Object::bind() const or unbind() functions. Instead, the QueryObject uses the QueryObject::begin(const Enum) and QueryObject::end(const Enum) functions.
