void mc::gfx::QueryObject::begin ( const Enum &  target)

This QueryObject will begin querying data for a specified target.

Replacement for the Object::bind() const function.

The results can be queried with one of the QueryObject::get funcitons.

Parameters
targetTHe kind of data to query. Can not be GL_TIMESTAMP.
See also
https://www.opengl.org/wiki/GLAPI/glBeginQuery
QueryObject::counter()
Attention
When you use this function make sure to also call QueryObject::end(const Enum&) eventually
This uses an OpenGL function and must be called in a thread with an OpenGL context. Otherwise, an error will be thrown.
Exceptions
GL_INVALID_OPERATIONIf the current thread does not have an OpenGL context
