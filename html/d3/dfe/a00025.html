Detailed Description

Represents an OpenGL Framebuffer Object which allows rendering to a non-screen location.

This also allows rendering to a Texture2D or RenderBuffer object.

This is not a subclass of the Buffer object as this is technically not an OpenGL sslBuffer. The name is misleading.

See also
https://www.opengl.org/wiki/Framebuffer_Object
