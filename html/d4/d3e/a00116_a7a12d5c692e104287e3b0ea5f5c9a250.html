bool mc::MACE::isRunning ( )

Checks whether the MACE is ready to be updated.

Will return true if MACE::init() has been called, and MACE::destroy() and MACE::requestStop() have not been called.

Returns
If update() should be called. If this returns false, you should exit the main loop and call destroy()
