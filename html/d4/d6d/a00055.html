Detailed Description

Stores vertex data for a `VertexArray.

This is absolutely crucial for any rendering inOpenGL.`

A VertexArray is simply an array of VertexBuffer behind the scenes. It provides an easy way to have multiple sets of data for one rendering pass, such as colors, texture coordinates, and normals.

See also
https://www.opengl.org/wiki/Vertex_Specification::Vertex_Buffer_Object
