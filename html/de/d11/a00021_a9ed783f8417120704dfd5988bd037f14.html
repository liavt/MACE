virtual void mc::gfx::Entity::onDestroy ( )
protectedpure virtualinherited

When Entity.destroy() is called, onDestroy() is called on all of it's children.

See also
MACE::destroy()
Warning
This should only be used internally or by advanced users. Misuse can cause undefined behavior
Attention
This uses an OpenGL function and must be called in a thread with an OpenGL context. Otherwise, an error will be thrown.
Exceptions
GL_INVALID_OPERATIONIf the current thread does not have an OpenGL context

Implemented in mc::gfx::Group, mc::gfx::Button, mc::gfx::Text, mc::gfx::Letter, mc::gfx::ProgressBar, and mc::gfx::Image.
