virtual void mc::gfx::ogl::Object::bindIndex ( const Index  id) const
protectedpure virtual

Intended to be overloaded to bind to a certain index.

Object::bind() const and Object::unbind() const both use this function.

Parameters
idIndex to bind to. If it is 0, it needs to be unbinded.
Attention
This uses an OpenGL function and must be called in a thread with an OpenGL context. Otherwise, an error will be thrown.
Exceptions
GL_INVALID_OPERATIONIf the current thread does not have an OpenGL context
Warning
This should only be used internally or by advanced users. Misuse can cause undefined behavior
