virtual void mc::gfx::Entity::init ( )
protectedvirtualinherited

Should be called a by Entity when MACE.init() is called.

Calls onInit()

Overriding this function is dangerous. Only do it if you know what you are doing. Instead, override onInit()

Note
This will make this Entity dirty.
See also
Entity::DIRTY
Attention
This uses an OpenGL function and must be called in a thread with an OpenGL context. Otherwise, an error will be thrown.
Exceptions
GL_INVALID_OPERATIONIf the current thread does not have an OpenGL context
InitializationErrorIf the property Entity::INIT is true, meaning init() has already been called.

Reimplemented in mc::gfx::GraphicsEntity, and mc::os::WindowModule.
