void mc::gfx::PointerComponent::destroy ( )
finalprotectedvirtual

Called when Entity::destroy() is called or the Component is removed via Component::update(Entity*), whichever comes first.

Once Component::destroy(Entity*) is called, it is immediately removed from the Entity. Required function.

Attention
This uses an OpenGL function and must be called in a thread with an OpenGL context. Otherwise, an error will be thrown.
Exceptions
GL_INVALID_OPERATIONIf the current thread does not have an OpenGL context
